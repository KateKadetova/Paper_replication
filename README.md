# Репликация Chen and Li (2009)
# Группа Кадетова, Новикова, Худокормова

Групповой проект по курсу «Экспериментальные методы в общественных науках».

Статья: Chen, Y., & Li, S. X. (2009). *Group Identity and Social Preferences*. American Economic Review, 99(1), 431–457.

## Что мы реплицируем

В проекте воспроизводятся основные результаты из текста статьи:

- Table 1 — описание экспериментальных сессий;
- Figure 1 — распределение токенов между другими участниками в первоначальном тритменте;
- Table 2 — параметры распределительных предпочтений;
- Table 3 — логит-регрессии для положительной и отрицательной взаимности;
- Table 4 — доля решений, максимизирующих общественное благосостояние;
- Table 5 — ожидаемые и реальные выигрыши;
- Table 6 — доля участников, которые меняют решения в зависимости от внутригруппового и межгруппового дизайна;
- Table 7 — влияние элементов дизайна на самооценку привязанности к группе.

Таблицы из прложения не реплицируются.

## Файлы

Для запуска нужны файлы:

- `replication.R` — основной R-скрипт;
- `20061107_data_behavior_Chen_Li.dta` — поведенческие данные эксперимента;
- `20061107_data_survey_Chen_Li.dta` — данные пост-экспериментального опроса.

## Как запустить

1. Положить два `.dta`-файла в папку `/data`.
2. В начале `replication.R` проверить пути:

```r
behavior_path <- "/data/20061107_data_behavior_Chen_Li.dta"
survey_path   <- "/data/20061107_data_survey_Chen_Li.dta"
```

3. Запустить весь скрипт.

Если каких-то R-пакетов нет, скрипт установит их автоматически.

## Что получается на выходе

Скрипт создает папку `outputs/` и сохраняет туда:

- `figure1_other_other_allocations.png`;
- `replication_tables.xlsx`;
- отдельные `.csv`-файлы для таблиц.

Также таблицы печатаются в консоль при запуске скрипта.
